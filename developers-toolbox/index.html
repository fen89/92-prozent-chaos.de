# VSCode - der Code-Editor

Lange Rede, kurzer Sinn: Ohne VS Code geht im Frontendbereich bei mir gar nichts - ist & bleibt für mich der beste Code-Editor der letzten Jahre. Die Fülle an Funktionen, die vielen Bugfixes und Updates, und natürlich die Massen an Plugins - wunderbar!
